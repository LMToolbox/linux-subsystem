# What's that?

It's Linux subsystem and proot builder.
Builds alpine-based subsystem with packages from pkgs.txt (and auto-updates) and also proot (based on versions specified in `proot/config.sh`). All the generated files are ready-to-paste into jniLibs in Android Studio app project.

## Releases

Releases are in the releases tab, with tags `YY.MM.DD`, and for proot binaries `proot-*`.
